📁 Option A — CSV Import Easiest

Export MySQL data to CSV files, import them into Power BI Desktop. Best for demos and presentations.

python powerbi/export_for_powerbi.py

Creates 7 pre-aggregated CSVs in powerbi/data/

🔗 Option B — Live MySQL Live Data

Connect Power BI directly to your MySQL database for real-time refresh. Requires MySQL ODBC Connector 8.x (64-bit).

Server: localhost   Database: hr_analytics

1

Install Power BI Desktop (Free)

Download from powerbi.microsoft.com/desktop. Sign in with a Microsoft work/school account (free). Personal Microsoft accounts also work for Publish to Web.

2

Export Data from MySQL

Run: python powerbi/export_for_powerbi.py — generates 7 CSV files for all charts. Then in Power BI Desktop: Home → Get Data → Text/CSV and import each file from powerbi/data/.

3

Build 8 Charts (see CHARTS_GUIDE.md for DAX + settings)

Create the 8 visuals shown below. The guide at powerbi/CHARTS_GUIDE.md has exact visual types, field mappings, DAX measures, and color codes for each chart.

4

Publish to Power BI Service

In Power BI Desktop: Home → Publish → select your workspace. Open app.powerbi.com to confirm your report appeared.

5

Get Your Embed URL

Public embed (recommended for demo): In app.powerbi.com, open your report → File → Publish to web (public) → Create embed code → copy the page URL (not the iframe code).

Secure workspace embed: File → Embed report → Website or portal → copy the src URL from the generated iframe.

6

Paste URL Below → Done!

Once saved, the Power BI Report tab in your HR Dashboard will immediately show your live report — fully embedded inside Flask with no page redirect.
